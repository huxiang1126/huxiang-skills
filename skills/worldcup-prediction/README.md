# worldcup-prediction

2026 世界杯赛前盘口预测总控 Skill。

## v2.7.1：相对最优盘口版

这版把输出定位改成更适合球迷使用的盘口协助框架：默认给出一个相对最优可执行方向，不再把“盘口不碰”作为常规结论。

```text
联网核实资料
-> 18-Agent 证据回执
-> 无盘口独立足球判断
-> 总预测官五面判断
-> 相对最优主推盘口
-> 朋友圈文案 + 两张图
-> 写入本地账本
```

核心变化：

- 回溯只做审计提醒，不参与本场初始概率。
- 默认必须在让球、受让、大小球里选一个相对最优方向。
- 薄边际用「可博 / 0.25U」表达，不用“不投注”结束。
- 只有赛程、盘口、首发等关键信息无法核实时，才允许信息熔断不投注。
- 下盘/受让盘提高门槛，不能因为上盘贵就自动反买。
- 强队穿盘能力单独建模，热门方向可以成立。
- 大小球按多比分路径判断，不再被波胆低命中率误伤。
- 保留 18-Agent 回执、盘口来源、首发/天气、两张图、京華老宋体长图、U 单位和波胆纪律。

## 输出结构

固定五部分：

1. 最终预测结果
2. 3 个比分方案
3. 盘口投注建议
4. 朋友圈文案
5. 朋友圈海报图 / 分析思路图

## 总预测官五面判断

1. **赛果面**：不看盘口，先判断真实比赛更像谁赢、平局还是爆冷。
2. **穿盘面**：单独判断强队有没有早进球、边路爆点、高点、定位球、替补火力和净胜球动机。
3. **受让面**：下盘必须证明自己有真实武器和抗压路径，不能只靠“上盘贵”。
4. **进球数面**：大球/小球必须由多个比分路径共同支持。
5. **价格面**：盘口只回答值不值得买，不替代足球判断。

## 投注纪律

- 最终必须给一个主推盘口方向。
- 主推可以是让球盘、受让盘或大小球。
- 不投注只作为信息熔断，不作为常规价格判断。
- `U=用户自定娱乐单位，不是固定金额。`
- 波胆只能小额可选；比分路径不等于波胆下注清单。
- 所有结论仅供娱乐参考，不保证赛果。
